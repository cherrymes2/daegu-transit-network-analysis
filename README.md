# Daegu Metro Network Analysis

## Overview

This repository contains research on the impact of metro network reconfiguration on travel patterns in Daegu, South Korea.

## Research Topics

* Metro network centrality analysis
* Transfer station effects
* Origin-Destination (OD) flow changes
* Mobility hub integration
* Demand Responsive Transport (DRT)

## Methods

* Network Analysis
* NetworkX
* GIS Analysis
* Interrupted Time Series Analysis
